--How to make a server on your PC:--
Click the fabric-installer-0.7.4 and choose 1.17.1 and install server 0.11.6 into a new folder > Download server jar > Press Generate > Done > Exit
> Drag Mods Folder, Configs Folder, start.bat, server-icon, server.properties and kubejs folder into the new folder you made 
> run the fabric-server-launch (If this comes up with an error it means you need Java 16)
> accept EULA agreement 
> launch the start.bat file
